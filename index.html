The difference between var and let includes:

  • Var can be redeclared while let cannot
  • Var is function scoped and let is block scope
  • var is not ES2015 and let is

The difference between var and const includes:

  • Var can be redeclared and reassigned while const cannot
  • Var is function scoped and const is block scope
  • var is not ES2015 and const is

The difference between const and let includes:

  • let can be reassigned while const cannot
  • const cannot be declaired without a value and let can

Hoisting is how we explain the complier. Variables are lifted to the top of the scope they are declared in. Functions work in a similar way and can be called before they are defined in a codebase if they are defined further down.
